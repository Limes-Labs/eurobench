# EuroBench v0.4 public hard-mode shard

This directory contains a compact public v0.4 shard generated by:

```bash
python3 scripts/generate_v04_tasks.py --seed public-v0.4 --output tasks/v0.4/hard_public.json
```

The shard has 12 synthetic tasks across European law/admin boundaries, multilingual cross-lingual reasoning, long-context contradiction handling, safe cybersecurity assessment, math/physics reasoning, and agentic/tool-use planning.

v0.4 is intentionally small. Its purpose is to exercise a richer review contract:

- source-grounded evidence and distractor rejection
- partial-credit scoring dimensions
- critical failures for fluent but unsafe or unsupported answers
- explicit contamination assumptions through result cards

The public shard is not a leaderboard. Use held-out private variants and human review before making model-quality claims.
