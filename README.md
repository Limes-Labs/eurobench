# EuroBench

Benchmark plan and initial tasks for European languages and institutional use cases.

Limes Labs will not claim model quality without measurement. EuroBench starts with small, reproducible evaluation work: reasoning, coding, translation, law, public administration, education, and safety — across Italian, French, German, Spanish, Polish, Dutch, Greek, Portuguese, Romanian, and other European languages.

## Goals

- Identify benchmark gaps for European languages and institutions
- Define reproducible evaluation tasks with clear limitations
- Test open weight models with published methodology
- Publish results, including failures
